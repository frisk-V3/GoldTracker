# GoldTracker

金（ゴールド）価格を定期的に監視・記録し、投資判断の基礎データを蓄積するためのバックグラウンドサービスです。短期的なボラティリティの把握と、中長期的な価格推移のログ保存を目的としています。

## 投資支援機能

本ツールは単なる記録だけでなく、以下の投資分析サポートを行います。

1. **ベースラインの把握（3日間集中学習）**
   稼働開始から72時間を「学習期間」とし、その期間内の平均価格・最高値・最安値を算出します。これにより、現在の価格が相対的に高いのか低いのかを判断する指標を提供します。

2. **継続的なエビデンス蓄積**
   CSV形式で全データを保存し続けるため、スプレッドシート等に取り込んで独自のテクニカル分析やチャート作成を行うことが可能です。

3. **リアルタイム・スプレッド監視**
   Ask（買値）とBid（売値）を同時に記録することで、取引コスト（スプレッド）の変動を可視化します。

## 主な仕様

* プロジェクト名: GoldTracker
* 取得間隔: 14.5分（1日約100回のサンプリング）
* 参照ソース: GoldAPI.io (XAU/USD または指定通貨)

## セットアップと運用

### 1. APIキーの設定
Worker.cs 内の `ApiKey` 定数に有効なトークンを設定してください。

### 2. 学習データの活用
3日経過後に生成される `study_result.json` を確認してください。
* `Fluctuation`: 期間内の値動き幅を確認し、エントリータイミングの参考にします。
* `AveragePrice`: 平均値からの乖離を確認し、割安・割高の判断材料にします。

## 投資上の注意点

* **スリッページと手数料**: 本ツールで取得する価格は市場仲値ベースである場合があります。実際の取引所の価格やスプレッドとは異なる可能性があるため、必ず取引画面と照合してください。
* **データ遅延**: ネットワーク環境やAPI側の制限により、リアルタイム価格と数秒〜数分の乖離が発生する場合があります。秒単位のスキャルピングには適しません。
* **リスク管理**: 本ツールは過去のデータ集計を行うものであり、将来の利益を保証するものではありません。
